Detailed Action Plan: Web Development (Calendar Project)

Skill Focus

Build your VERY OWN Calendar! Following the steps below:

Overall Aim

To develop a functional and visually structured calendar webpage using HTML for content structure and CSS for styling, while progressively improving my understanding of web development principles such as layout design, data organisation, and basic user interface presentation.

Phase 1: Planning & Skill Preparation (Day 1)

  • Identify the purpose of the project: create a simple monthly calendar webpage.
  • Decide on structure: use HTML <table> as the base layout system.
  • Identify required elements: table, tr, th, td
  • Apply basic CSS styling for clarity

Timeframe: 1 day

Milestone: Clear understanding of layout before coding

Phase 2: Structure Development (Day 2–3)

  • Create HTML structure (DOCTYPE, html, head, body)
  • Build table layout
  • Add title row using colspan
  • Create weekday headers (Mon–Sun)
  • Insert date rows

Timeframe: 2 days

Milestone: Calendar grid visible in browser

Phase 3: Functional Completion (Day 4)

  • Fill in all dates correctly
  • Align dates under correct weekdays
  • Fix structure and spacing issues

Timeframe: 1 day

Milestone: Fully working calendar

Phase 4: Styling & Visual Design (Day 5–6)

  • Center table using margin auto
  • Add borders and padding
  • Align text to center
  • Style header with dark background and white text
  • Improve font readability

Timeframe: 2 days

Milestone: Clean, readable layout

Phase 5: Testing & Final Adjustments (Day 7)

  • Test in browser
  • Fix alignment and spacing issues
  • Ensure consistent styling

Timeframe: 1 day

Milestone: Final polished project

This action plan demonstrates a structured and progressive approach to developing web development skills through a real-world practical task. It includes planning, implementation, testing, and refinement, ensuring measurable skill growth.
